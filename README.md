Full Stack React + Django E-Commerce Project

Backend
- Django REST Framework
- PostgreSQL or SQLite
- JWT Auth
- requirements.txt in `/backend`

Frontend
- React + Bootstrap/Material UI
- Axios for API calls
- `/frontend` directory

Installation:-

Frontend:
bash:
cd frontend
npm install
npm start

Backend:
bash:
cd backend
pip install -r requirements.txt
python manage.py runserver
